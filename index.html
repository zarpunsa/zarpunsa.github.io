Saya seorang mahasiswa di Institut Teknologi PLN

Perkenalkan nama saya Muhammad Rizky Bamby Saputra akrab disapa Iky. Saya lahir di Jakarta pada tanggal 30 Agustus 2003. Saat ini saya ber-status sebagai mahasiswa aktif Institut Teknologi PLN dari jurusan Sistem Informasi. Saat ini saya bertempat tinggal di Duren Sawit, Jakarta Timur. Kesibukan saya adalah selain sebagai mahasiswa saya aktif sebagai ketua Forum Komunikasi di jurusan saya. Saya orang yang gemar bermain game online bersama pacar saya dan bermain futsal bersama teman-teman saya di waktu luang saya.
